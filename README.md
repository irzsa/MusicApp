MusicApp was created to provide a simple and efficient solution for managing song collections and to assist musicians in their creative and learning processes. Whether you are a passionate amateur musician or a dedicated professional, MusicApp provides the tools you need to manage and improve your musical practice.


---

The main interface includes an organized table containing the details of each song in your collection (key, BPM, artist's name, and song name). 
* **Add:** Fill in the necessary information and press "Add".
* **Delete:** Click on the corresponding row and press "Delete".
* **Search:** A dynamic search bar allows you to filter data by selecting a specific search criterion.

Technologies and Libraries Used

* **Language:** C#
* **Framework:** .NET Platform
* **UI:** Windows Forms (WinForms) for a familiar and easy-to-understand user experience.
* **Audio:** Integrated sound libraries to generate precise rhythmic metronome ticks based on user-specified BPM.
